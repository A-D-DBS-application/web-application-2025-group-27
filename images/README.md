# Images Directory

This directory contains visual assets and documentation for the Rival project.

## Structure

### 1. App Screenshots (`app-screenshots/`)
Screenshots of the actual running application:

- `landing.png` — Landing/homepage of the application
- `the_team.png` — Team members page

### 2. UI Prototype (`ui-prototype/`)
Screenshots and images of the UI prototype generated by Lovable. This includes:

- **Dashboard/Homepage**: Main intelligence workspace dashboard
- **Login Page**: User authentication interface
- **Signup/Create Workspace**: User registration and workspace creation forms
- **Company Snapshot**: Company overview page
- **Company Detail**: Detailed company information page
- **Competitors/Competitive Landscape**: Competitor listing and relationship views
- **Industries/Industry Signals**: Industry tags and signals page
- **Products**: Product listing and detail pages
- **Contacts/Team Members**: Contact and team member profiles
- **Key Contacts**: Key contacts listing page

### 3. User Stories (`user-stories/`)
User stories documentation in PDF format.

- `User Stories.pdf` — Complete user stories documentation

### 4. Database (`database/`)
Database schema diagrams and screenshots.

- Database schema ERD/diagram screenshots
- Shows the complete schema including:
  - `companies` table
  - `company_industry` join table
  - `company_competitor` join table
  - `industries` table
  - Relationships between tables

## Notes

- App screenshots should be updated when significant UI changes are made
- UI prototype images are from the Lovable preview
- Database diagrams should reflect the current schema
